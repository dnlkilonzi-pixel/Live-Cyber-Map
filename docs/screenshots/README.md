# Screenshots

Place screenshots of the application here. The following files are referenced in the root README:

| File | Description |
|------|-------------|
| `overview.png` | Full-screen view of the globe with active attack arcs |
| `dashboard.png` | Close-up of the left-side stats dashboard panel |
| `attack-feed.png` | Close-up of the right-side live attack feed |
| `anomaly-alert.png` | Globe with the red anomaly detection alert banner visible |
| `replay-mode.png` | Replay controls UI with speed control slider |

## How to capture screenshots

1. Start the app: `docker-compose up --build` (or `docker compose up --build` on Docker v2)
2. Open `http://localhost:80` in a browser
3. Wait ~30 seconds for the globe to populate with live attack arcs
4. Use your OS screenshot tool or browser DevTools to capture each view
5. Save the files here using the names listed above
